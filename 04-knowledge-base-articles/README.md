# Knowledge Base Articles

This folder contains sample knowledge base articles written in Markdown. KB articles are designed for end users — clear, jargon-minimal, and structured for quick scanning.

## Articles in this folder

| File | Topic |
|------|-------|
| `clear-browser-cache-and-cookies.md` | How to clear cache and cookies across Chrome, Firefox, Edge, and Safari |

## Document structure

Each KB article uses the following structure:

1. **What it is** — Brief explanation of the concept (where relevant)
2. **When to use this** — Scenarios that trigger the need for this article
3. **How-to steps** — Organised by product or platform
4. **What to expect after** — Setting expectations post-action
5. **Related articles** — Cross-linking for discoverability

This format is consistent with industry-standard KB systems including Zendesk, Confluence, Freshdesk, and ServiceNow.
