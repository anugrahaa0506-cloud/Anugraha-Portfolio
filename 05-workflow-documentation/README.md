# Workflow Documentation

This folder contains sample workflow documentation demonstrating how to describe end-to-end processes for technical and operational teams.

## Documents in this folder

| File | Topic |
|------|-------|
| `multi-region-content-publishing-workflow.md` | End-to-end content publishing workflow for multi-region documentation operations |

## What this sample demonstrates

- Scope definition (what's in and out of scope)
- Roles and responsibilities table (RACI-adjacent)
- Stage-by-stage workflow breakdown with owners, steps, and outputs
- Escalation and error handling matrix
- SLA reference table by content type

## Context

This sample is modelled on the type of workflow documentation used in enterprise content operations — the format is applicable to teams using platforms like Confluence, SharePoint, or any internal documentation system.
