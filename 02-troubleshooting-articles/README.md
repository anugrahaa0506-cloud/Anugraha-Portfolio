# Troubleshooting Articles

This folder contains structured troubleshooting articles written in Markdown. Each article follows the same format used in enterprise technical support documentation — symptom, possible causes, step-by-step resolution, expected result, and escalation path.

## Articles in this folder

| File | Product | Topic |
|------|---------|-------|
| `airpods-not-connecting-iphone.md` | Apple AirPods | Bluetooth connectivity failure on iPhone |
| `wifi-network-not-showing-windows11.md` | Windows 11 | Wi-Fi network not appearing in available networks list |

## Document structure

Each article uses the following structure:

1. **Symptom** — What the user is experiencing
2. **Possible causes** — Ranked table of likely root causes
3. **Resolution steps** — Numbered, sequential troubleshooting steps
4. **Expected result** — What success looks like
5. **Escalation** — What to do if the issue persists

This structure mirrors the format used in enterprise knowledge base systems and is compatible with DITA task and troubleshooting topic types.
