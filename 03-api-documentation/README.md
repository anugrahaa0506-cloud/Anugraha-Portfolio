# API Documentation

This folder contains a sample API documentation set written for the OpenWeatherMap REST API, using their publicly available free-tier API.

## Files in this folder

| File | Description |
|------|-------------|
| `openweathermap-api-guide.md` | Complete getting started guide and endpoint reference |

## What this sample demonstrates

- Getting started guide structure (prerequisites, authentication, first request)
- Response field documentation with a full JSON example and annotated table
- Endpoint reference with parameter tables (required vs optional, types, descriptions)
- Units of measurement reference
- Error code documentation
- Rate limit information

## Audience

This documentation is written for developers integrating weather data into applications. It assumes basic familiarity with REST APIs but does not assume knowledge of the OpenWeatherMap platform specifically.

## Tools and standards used

- Markdown
- REST API documentation conventions (modelled after Stripe, Twilio, and Postman documentation style)
- GitHub-flavoured Markdown tables for parameter references
